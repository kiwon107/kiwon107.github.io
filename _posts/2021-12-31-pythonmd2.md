## 2-1. Immutable & Mutable

Mutable 객체: 객체가 지닌 값의 수정이 가능한 객체<br/>
Ex) 리스트, 딕셔너리

```python
today = ['공부하자', '운동하자']
id(today)
```
    (결과) 1896625928712




```python
today += ['요리하자', '일찍자자']
id(today)
```
    (결과) 1896625928712

Immutable 객체: 객체가 지닌 값의 수정이 불가능한 객체<br/>
Ex) 튜플, 문자열

```python
today = ('공부하자', '운동하자')
id(today)
```
    (결과) 1896629021000

보다시피, 튜플은 저장된 값을 수정하면 새로운 튜플이 만들어진다.
